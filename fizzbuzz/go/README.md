# Usage
### Without Build
```
go run main.go -count=<int> -print=<bool>
-count is optional. Default: 100
-print is optional. Default: true
```
### With Build
```
go build
./goFizzBuzz -count=<int> -print=<bool>
-count is optional. Default: 100
-print is optional. Default: true
```

# Examples
```
go run main.go -count=10000 -print=false
go run main.go -count=200
go run main.go
```
